Tweet-A-Fact

Puts a spin on "internet trolling." Instead of tweeting insults or critiques to politicians we don't appreciate, this bot helps you send them interesting facts about dozens of different topics.

  • Video Demo | Repo
  • • Built with Ruby on Rails
  • • Integration with Twitter API to generate and retrieve tweets
  • • Integration with facts API to generate list of facts for user to choose from and then display a fact at random by category
  • • User clicks a button to have bot send out the tweet
  • • Database has a list of politicians and each tweet is sent out to one of them at random
